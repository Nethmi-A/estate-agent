# Project Overview

this project is a responsive Estate Agent Web Application developed using React. It allows users to search for properties using multiple criteria, view detailed property information, browse image galleries, floor plan of the property and the location. It also manages a list of favourite properties.

This application has been developed as part of coursework requirement of the Advanced Client-Side Development module and is deployed as a live web application
